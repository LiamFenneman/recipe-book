# Leptos CSR Template

### Quickstart
1. `git clone https://github.com/LiamFenneman/leptos-csr.git`
1. `cd leptos-csr`
1. `cargo install trunk`
1. `trunk serve` or `trunk serve --open`
1. Open `http://localhost:8080`

### Features
- Minimal Leptos configuration
- TailwindCSS pre-configured
